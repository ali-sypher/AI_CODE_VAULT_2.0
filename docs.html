Platform Documentation

AI Code Vault 2.0 is a comprehensive AI-powered code repository analysis platform. It integrates automated repository scanning, GenAI-driven code classification (hubs, links, satellites), and vector-based semantic search to deliver an enterprise-grade developer experience.

Core Architecture

1. Repository Scanning & Embedding

The platform autonomously recursive scans any provided repository. Each Python file is systematically broken down using our Abstract Syntax Tree (AST) engine to identify methods, classes, and logic boundaries. These logic blocks are then embedded using state-of-the-art vector models and stored in our ChromaDB vector space.

2. Hybrid Semantic Search

Gone are the days of manual logic tracking. The search architecture blends precise TF-IDF keyword matching with deep vector Cosine-Similarity analysis. This guarantees instantaneous retrieval of code blocks, whether you query by exact variable naming or conceptual logic abstraction.

3. Vault Architect (RAG LLM)

An integrated, highly conversational GenAI Chat Assistant. The Architect has full contextual access to the vector database. When queried, it retrieves your exact code snippets directly into the LLM context, acting as a pair-programmer with absolute knowledge of your codebase.
